# TypeScript Practice Programs

## 📌 Project Description

This repository contains basic TypeScript programs created for learning and practice.
It includes examples of variables, functions, interfaces, classes, abstract classes, and generics.

## 🛠 How to Install TypeScript

Run the following command in terminal:

npm install -g typescript

## ▶ How to Compile TypeScript File

tsc filename.ts

Example:

tsc variables.ts

## ▶ How to Run JavaScript File

node filename.js

Example:

node variables.js

## 📂 Programs Included

* Variables and Data Types
* Functions
* Interface
* Classes and Objects
* Abstract Class
* Generics
